Sobre Este Projeto

Um sistema de login moderno e responsivo

📄

HTML

O HTML (HyperText Markup Language) é a estrutura base do projeto. Utilizamos:

  • Elementos semânticos para melhor acessibilidade
  • Formulários e inputs validados
  • SVG para gráficos vetoriais (avatar do usuário)
  • Meta tags para responsividade
🎨

CSS

O CSS (Cascading Style Sheets) é responsável pelo visual e design. Características incluem:

  • Gradientes e cores coordenadas (azul e preto)
  • Animações suaves (slideIn, slideInLeft)
  • Design Responsivo (mobile, tablet, desktop)
  • Efeitos hover e transições
  • Flexbox para layout flexível
⚙️

JavaScript

O JavaScript adiciona interatividade ao projeto. Funcionalidades:

  • Validação de formulários
  • Verificação de senha (12345678)
  • Redirecionamento após login
  • Manipulação do DOM
  • Tratamento de eventos
