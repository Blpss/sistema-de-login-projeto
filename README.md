# sistema-de-login-projeto
# 🔐 

Este projeto é um sistema de cadastro e autenticação de usuários desenvolvido com foco em praticar conceitos fundamentais do desenvolvimento web.

## 🚀 Funcionalidades

- Cadastro de usuários
- Validação de dados no formulário
- Sistema de login
- Recuperação de senha
- Interface moderna e responsiva

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de aprimorar meus conhecimentos em lógica de programação, manipulação de DOM e validação de formulários, simulando o funcionamento real de um sistema de autenticação.
